# Task 1: Data Cleaning & Preprocessing

## Objective
To clean and preprocess the Titanic dataset for machine learning.

## Steps Performed
1. Loaded dataset
2. Checked dataset information
3. Handled missing values
4. Removed Cabin column
5. Encoded categorical variables
6. Visualized outliers using boxplots
7. Removed outliers using IQR method
8. Standardized numerical features
9. Saved cleaned dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

Titanic Dataset
